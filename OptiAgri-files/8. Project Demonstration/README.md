## How to Demo Live
1. Run the app (see `5. Project Development Phase/README.md`).
2. Open `http://127.0.0.1:5000` — the home page shows the prediction form.
3. Try a sample profile:
   - Rice-like: N=90, P=42, K=43, temperature=24.8, humidity=82, ph=6.5, rainfall=210
   - Banana-like: N=105, P=82, K=52, temperature=27.5, humidity=80, ph=6.1, rainfall=105
4. Show the result page — crop recommendation, confidence %, soil health
   score, fertilizer/season/water guidance.
5. Visit `/dashboard` to show model comparison charts and EDA visuals.
6. Visit `/about` for the project description page.
